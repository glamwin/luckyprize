# LuckyPrize

Простой фронтенд-проект «Колесо фортуны» без серверной логики.

## Что реализовано

- Современный интерфейс в одном экране.
- Виртуальный баланс (старт: `1000 ₽`).
- Кнопка вращения колеса.
- Случайный выбор сектора и зачисление выигрыша на баланс.
- Список секторов и наград.

## Локальный запуск

Открой `index.html` в браузере или запусти статический сервер:

```bash
python3 -m http.server 4173
```

После этого открой `http://localhost:4173`.

## Деплой на GitHub Pages

1. Запушь репозиторий на GitHub.
2. Перейди в `Settings` → `Pages`.
3. В разделе **Build and deployment** выбери:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (или текущая ветка) / `/ (root)`
4. Сохрани настройки и подожди публикацию.

Сайт будет доступен по адресу вида:
`https://<username>.github.io/<repo>/`
